# BudgetWise 💰

BudgetWise هو تطبيق بسيط ومرن لتتبع الميزانية يسمح لك بإضافة، عرض، وحذف الدخل والمصروفات في الوقت الفعلي. يتم حفظ البيانات باستخدام Firebase Realtime Database.

## 🚀 المميزات

- إضافة دخل ومصروفات مع الوصف والتاريخ.
- عرض المعاملات في الوقت الفعلي.
- حساب الرصيد، إجمالي الدخل، وإجمالي المصروفات.
- حذف المعاملات مع نافذة تأكيد (SweetAlert2).
- إشعارات نجاح أو خطأ باستخدام Toastify.
- واجهة استخدام مرنة وسهلة.
- مستضاف عبر GitHub Pages.

## 🛠️ التقنيات

- HTML5 / CSS3
- JavaScript (ES6+)
- Firebase Realtime Database
- Webpack
- Toastify JS
- SweetAlert2

## 📦 التثبيت

1. استنساخ المشروع:

   ```bash
   git clone https://github.com/your-username/budgetwise.git
   cd budgetwise

   ```

2. تثبيت التبعيات:

npm install

3. تشغيل الخادم المحلي:

npm start

🔥 إعداد Firebase
قم بتحديث ملف firebase.js ببيانات مشروعك وتأكد من تفعيل قاعدة البيانات الفورية.

🧪 التجربة
المعاملات يتم حفظها وتحميلها عبر Firebase.

Toastify يظهر إشعارات بالنجاح أو الخطأ.

SweetAlert2 يظهر نافذة تأكيد عند الحذف.
